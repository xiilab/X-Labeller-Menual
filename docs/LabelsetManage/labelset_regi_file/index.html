라벨셋 일괄 등록

1. 템플릿 다운로드 버튼 클릭 시 라벨셋 등록 템플릿 파일이 다운로드 됩니다.​

2. 파일 첨부 버튼 클릭 후, 작성한 라벨셋 등록 템플릿 파일을 첨부 할 수 있습니다.​

  • 엑셀 파일만 첨부 가능합니다.
  • 다운로드 받은 템플릿 양식과 동일하지 않은 경우 등록이 제한됩니다.

3. 일괄 삭제 버튼 클릭을 통해, 첨부한 파일 삭제가 가능합니다.​

4. 라벨셋 등록 템플릿 파일 첨부 영역​

  • 작성한 라벨셋 등록 템플릿 파일을 해당 영역에 드래그 앤 드롭하여 첨부가 가능합니다.
  • 엑셀 파일이 아닌 파일은 첨부되지 않습니다.

5. 등록 버튼을 클릭하여, 첨부한 템플릿 정보를 토대로 라벨셋이 생성됩니다.​
