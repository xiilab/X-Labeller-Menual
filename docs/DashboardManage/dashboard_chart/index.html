대쉬보드

사용자별 대쉬보드 옵션 설정​

대쉬보드 컨트롤​

1. 탭 버튼 클릭시 사용자 별 차트 출력 여부를 설정할 수 컨트롤 영역을 볼 수 있습니다.​

2. 각 차트 별 체크박스 선택을 통해, 사용자 별 출력할 통계정보를 설정할 수 있습니다.​

3. reset 버튼 클릭시, 기존에 설정되어 있던 사용자 대쉬보드 정보( 출력 여부 / 차트크기 / 차트 위치 )로 값이 초기화 됩니다.​

4. save 버튼 클릭시, 설정한 정보대로 사용자 대쉬보드 정보가 저장됩니다.​

차트 크기 조정 및 위치 이동​

  • 마우스 클릭 및 드래그 앤 드롭 을 통해 차트 크기및 위치값을 수정할 수 있습니다.

프로젝트 역할별 대쉬보드 통계​

관리자 대쉬보드​

1. 프로젝트 정보를 확인합니다.​

프로젝트 일정, 목표 수량 진척도, 업로드 데이터 량

2. 데이터 생산 지표를 확인합니다.​

x 축 : 날짜 / y 축 : 검수완료된 데이터 갯수

3. 데이터 작업 현황를 확인합니다.​

데이터 상태별 비율 ( 분배대기중, 작업중, 검수요청, 1차 검수완료, 2차 검수완료, 1차 반려, 2차 반려, 불용 )

4. 반려​

⏳ 준비중 입니다.

5. 라벨 분포 지도를 확인합니다.​

라벨별 분포를 트리맵으로 표현

6. 라벨별 목표 달성률를 확인합니다.​

라벨별 목표 달성율 ( ~ 100% 는 파란색, 101% ~ 빨간색으로 표시 )

7. 데이터 가공 및 검수 트렌드를 확인합니다.​

x축 : 날짜 / y 축 : 데이터 갯수 ( 파란색 : 검수완료 / 검정색 : 검수 요청 )

검수자 대쉬보드​

1. 데이터 작업 현황을 확인합니다.​

2. 전체 데이터 비교 표를 확인합니다.​

전체 데이터 수, 검수요청, 1차 검수완료, 2차 검수완료, 반려

작업자 대쉬보드​

1. 데이터 작업 현황을 확인합니다.​

2. 전체 데이터 비교 표를 확인합니다.​

전체 데이터 수, 작업중, 검수요청, 1차 검수완료, 2차 검수완료, 반려
