Slice + Merge









  • Slice 기능은 시작,끝 점을 찍을 P버튼과 잘라낼 영역을 그리는 D버튼이 있습니다.

  • 객체를 선택 후 잘라낼 객체의 시작점을 클릭합니다.

  • 버튼이 D로 바뀐걸 확인한 후 잘라낼 영역을 그립니다.

  • 그리기가 완료된 후 끝 점을 찾아 클릭하면 분리된 객체를 확인할 수 있습니다.


  • 추가적으로 끝점이 없을경우 위 그림처럼 외부영역을 클릭하여도 분리된 객체를 확인할 수 있습니다.




  • Merge기능은 서로 붙어있는 두개의 객체를 차례로 선택하여 하나의 객체로 만드는 기능입니다.

  • 페이지 왼쪽 ToolBox에서 Merge이미지를 클릭시 Merge기능이 활성화됩니다.

  • Merge버튼이 활성화된 상태에서 연속적으로 붙어있는 이미지를 차례로 선택합니다.(선택시 객체 색상이 뚜렸해집니다.)

  • 객체선택 완료 후 Merge버튼 클릭시 하나로 합쳐진 객체를 확인할 수 있습니다.


  • 추가적으로 연속적으로 붙어있는 객체 2개이상을 차례대로 선택하여 Merge기능을 사용하실 수 있습니다.
