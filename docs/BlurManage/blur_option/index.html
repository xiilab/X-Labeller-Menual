비식별화

비식별화 처리​

비식별화 팝업으로 이동​

1. 데이터셋 관리 페이지에서 비식별 버튼 클릭시 비식별화 처리 옵션 설정 팝업으로 이동합니다.​

비식별화 처리 옵션 설정 팝업​

1. 검색어 입력 후 조회 버튼 클릭 시 검색 조건에 해당하는 데이터셋 정보를 조회할 수 있습니다.​

검색 조건 : 데이터셋 이름

2. 비식별화 처리할 데이터셋을 클릭시, 하단 영역에 선택한 데이터셋이 추가됩니다.​

3. 페이징 버튼을 클릭하여, 다음/이전 페이지로 이동합니다.​

4. 한 페이지당 조회할 데이터셋 수를 설정합니다.​

5. 비식별화 처리 옵션을 설정 합니다.​

비식별화 처리 옵션
  • 비식별 대상 검출 : 비식별화되어야 할 영역을 자동으로 검출하여 해당 영역에 바운딩 박스가 생성됩니다.
  • 비식별 대상 검출 + 비식별 : 비식별화되어야 할 영역을 자동으로 검출하여 해당 영역에 바운딩 박스가 생성되고 블러처리 됩니다.
  • 비식별 : 비식별화되어야 할 영역을 자동으로 검출하여 해당 영역이 블러처리 됩니다. ( 바운딩 박스는 생성되지 않음 )

6. 비식별 대상 검출, 비식별 대상 검출 + 비식별 옵션 선택시, 검출 대상을 선택합니다.​

겁출 대상 : 얼굴, 번호판, 얼굴 + 번호판

7. 삭제 버튼 클릭시, 선택한 데이터셋이 제외됩니다.​

8. 설정 버튼 클릭시, 설정한 옵션 정보대로 비식별화가 진행됩니다.​

비식별화 결과 확인​

1. 저작도구 팝업에서, 탭 버튼 클릭 합니다.​

2. 비식별 모드 버튼 클릭 시, 비식별화 모드로 변경되고, 처리된 결과를 확인합니다.​

비식별화 영역 수동 생성​

1. 저작도구 팝업의 비식별 모드에서, 추가 버튼을 클릭합니다.​

2. 캔버스 영역에서 마우스 드래그 앤 드롭을 통해, 비식별 처리할 영역에 바운딩 박스를 생성합니다.​

3. 비식별 타입을 지정합니다.​

4. 저장 버튼을 클릭합니다.​

5. 비식별화 처리 옵션 설정 팝업에서 비식별 처리 유형 중 비식별을 선택하고, 실행 버튼을 클릭합니다.​

6. 비식별화 결과를 확인 합니다.​
