데이터셋 반려

1. 반려 하려는 데이터 상태를 선택합니다.​

2. 검색어(데이터셋 이름) 입력 후, 조회 버튼 클릭 시 검색조건에 해당한는 데이터셋 목록을 조회할 수 있습니다.​

3. 반려 하려는 데이터셋의 체크박스를 하나 이상 선택합니다.​

4. 페이징 버튼 클릭시 다음/이전 목록으로 이동이 가능합니다.​

5. 셀렉트 박스 선택시 설정한 조건에 따라 사용자 정보가 정렬됩니다.​

표시 갯수 : 10개, 30개, 50개

6. 반려 버튼 클릭 시 선택한 데이터셋의 데이터들 중 조건에 맞는 데이터들이 반려 처리 됩니다.​

프로젝트 최종 완료 차수별 반려 가능한 조건

ℹ 반려는 1차/2차 반려가 존재합니다.

  • 1차 반려 ( 검수 요청 ➡ 1차 반려 )
    • 선택한 데이터셋의 데이터들 중 검수 요청 상태의 데이터만 반려 가능합니다.
    • 선택한 데이터셋의 데이터들 중 1차 검수자가 지정된 데이터만 반려 가능합니다.
  • 2차 반려 ( 1차 검수 완료 ➡ 1차 반려 )
    • 선택한 데이터셋의 데이터들 중 1차 검수 완료 상태의 데이터만 반려 가능합니다.
    • 선택한 데이터셋의 데이터들 중 2차 검수자가 지정된 데이터만 반려 가능합니다.

💬 프로젝트 최종 완료 차수가

  • 1차인 경우에는 ( 최종 완료 차수 : 1차 검수 완료 ) → 1차 반려 만 가능합니다.
  • 2차인 경우에는 ( 최종 완료 차수 : 2차 검수 완료 ) → 1차 반려, 2차 반려 둘다 가능합니다.

❗️ 최종 완료 차수 상태의 데이터는 반려할 수 없습니다.
