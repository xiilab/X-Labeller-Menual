데이터셋 목록

1. 검색 조건 설정 후 검색어를 입력하고, 조회 버튼 클릭 시 검색 조건에 해당하는 데이터셋 정보를 조회할 수 있습니다.​

검색 조건 : 이름, 등록자

2. 설정한 조건에 따라 데이터셋 정보가 정렬 됩니다.​

정렬 조건 : 업데이트 최신 순, 업데이트 과거 순, 등록 최신 순, 등록 과거 순, 데이터 많은 순, 데이터 적은 순

3. 테이블 목록에서 삭제를 원하는 목록의 체크박스를 하나 이상 선택하고 선택 삭제 버튼 클릭 시, 데이터셋 정보가 삭제됩니다.​

4. 배분 버튼을 클릭할 시 데이터셋 배분 팝업으로 이동합니다.​

5. 회수 버튼을 클릭할 시 데이터셋 회수 팝업으로 이동합니다.​

6. 검수 버튼을 클릭할 시 데이터셋 검수 팝업으로 이동합니다.​

7. 반려 버튼을 클릭할 시 데이터셋 반려 팝업으로 이동합니다.​

8. 합계 버튼을 클릭할 시 데이터셋 합계 팝업으로 이동합니다.​

9. 데이터셋 업로드 버튼을 클릭할 시 데이터셋 등록 팝업으로 이동합니다.​

10. 데이터셋 정보를 확인하고, 해당 테이블 목록 행 클릭 시 데이터셋 수정 팝업으로 이동합니다.​

데이터셋 ID, 이름, 데이터 수, 미할당 데이터 수, 작업중/검수요청/1차 검수 완료/2차 검수완료/반려 데이터 수, 등록자, 업로드일, 업데이트 일

11. 데이터 목록 버튼을 클릭시 데이터 관리 페이지로 이동하고 해당 데이터셋에 포함된 데이터 목록으로 갱신됩니다.​

12. 페이지 버튼을 클릭하여, 다음/이전 페이지로 이동합니다.​

13. 한 페이지당 조회할 데이터셋 수를 설정합니다.​
